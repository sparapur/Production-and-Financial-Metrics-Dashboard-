# Production & Financial Metrics Dashboard

A Power BI dashboard built during my co-op at Creation Technologies to give program managers real-time visibility into financial and operational performance across the business.

## Overview

Program managers needed a single place to track how programs were performing financially — across revenue, contribution margin, and tariffs — without pulling data manually from multiple sources. This dashboard replaced that process.

It was deployed site-wide and used daily to support financial tracking across approximately **$200M in revenue**.

---

## Features

**Filters**
- Filter by customer (individual or all)
- Filter by date range using a date slicer

**Revenue Page**
- Actuals vs. Forecast vs. Plan — monthly trend line
- Quarter and full-year view for broader planning context

**Contribution Margin Page**
- Actuals vs. Forecast vs. Plan — monthly and quarterly
- Tracks margin % rather than absolute values for cross-program comparability

**Tariffs Page**
- KPI cards: total tariffs expense and recovery at a glance
- Monthly breakdown table of pending tariffs (customer-aware)
- Tariffs Pending donut chart: claimed vs. unclaimed
- Tariffs Expense and Recovery trend lines by month

## Data & Architecture

- Unified data model built from **5+ cross-functional data sources**
- Automated reporting workflows to reduce manual refresh effort
- Dynamic filters allow self-serve analysis without needing to request custom reports
